## linux-pipes-and-redirection-intermediate

**Summary:** Shell redirection and pipelines on Ubuntu (`>`, `>>`, `2>`, `|`).

### Import on Killercoda

1. Go to https://killercoda.com and sign in.
2. Create Scenario → Import from GitHub → `https://github.com/kubernauten/killercoda-scenarios`.
3. Select the `linux-pipes-and-redirection-intermediate` directory as the scenario folder.
