## kubernetes-dashboard

You can start this scenario on https://killercoda.com as follows:

1. Go to https://killercoda.com
2. Sign in (or create a free account).
3. Click "Create Scenario" at the top and choose "Import from GitHub".
4. Use this repository: `https://github.com/kubernauten/killercoda-scenarios`.
5. In the import dialog, select the `kubernetes-dashboard` directory as the scenario folder.
6. Save the scenario and start it via the "Start Scenario" button.

Alternatively, you can fork this repository and import it as a scenario in your own account.
