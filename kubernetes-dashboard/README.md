## kubernetes-dashboard

Dieses Szenario kannst du auf killercoda.com wie folgt starten:

1. Gehe zu https://killercoda.com
2. Melde dich an (oder registriere dich kostenlos).
3. Klicke oben auf "Create Scenario" und wähle "Import from GitHub".
4. Gib dieses Repository an: `https://github.com/<ORG_ODER_USER>/killercoda-scenarios`.
5. Wähle im Import-Dialog das Verzeichnis `kubernetes-dashboard` als Scenario-Ordner.
6. Speichere das Scenario und starte es über den "Start Scenario" Button.

Alternativ kannst du das Repo forken und in deinem eigenen Account als Scenario importieren.
