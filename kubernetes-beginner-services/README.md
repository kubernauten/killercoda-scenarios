## kubernetes-beginner-services

[![Start on Killercoda](https://img.shields.io/badge/Start_on-Killercoda-2496ED?style=for-the-badge)](https://killercoda.com/kubernauten/scenario/kubernetes-beginner-services)

**Summary:** ClusterIP Service and in-cluster HTTP with DNS on a single-node cluster.

### Import on Killercoda

1. Go to https://killercoda.com and sign in.
2. Create Scenario → Import from GitHub → `https://github.com/kubernauten/killercoda-scenarios`.
3. Select the `kubernetes-beginner-services` directory as the scenario folder.
