### Import on Killercoda

1. Go to https://killercoda.com and sign in.
2. Create Scenario → Import from GitHub → `https://github.com/kubernauten/killercoda-scenarios`.
3. Select the `kubernetes-jobs-cronjobs-intermediate` directory as the scenario folder.
