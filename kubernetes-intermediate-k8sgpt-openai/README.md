# Kubernetes: K8sGPT with OpenAI

[![Start on Killercoda](https://img.shields.io/badge/Start_on-Killercoda-2496ED?style=for-the-badge)](https://killercoda.com/kubernauten/scenario/kubernetes-intermediate-k8sgpt-openai)

Install [K8sGPT](https://k8sgpt.ai/), connect the OpenAI backend with your own API key, deploy a deliberately broken Pod, and run `k8sgpt analyze` with optional `--explain`.

See `index.json` for the Killercoda scenario definition.
