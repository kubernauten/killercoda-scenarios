## linux-file-permissions-beginner

**Summary:** Practice numeric `chmod`, `ls -l`, and `stat` on files and directories (Ubuntu).

**Audience:** Beginners who completed an intro shell lab or know basic `cd`/`touch`.

### Import on Killercoda

1. Go to https://killercoda.com and sign in.
2. Create Scenario → Import from GitHub → `https://github.com/kubernauten/killercoda-scenarios`.
3. Select the `linux-file-permissions-beginner` directory as the scenario folder.
