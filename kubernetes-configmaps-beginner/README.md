**Summary:** ConfigMap literals and environment variable injection into a Deployment.

### Import on Killercoda

1. Go to https://killercoda.com and sign in.
2. Create Scenario → Import from GitHub → `https://github.com/kubernauten/killercoda-scenarios`.
3. Select the `kubernetes-configmaps-beginner` directory as the scenario folder.
