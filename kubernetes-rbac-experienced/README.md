## kubernetes-rbac-experienced

[![Start on Killercoda](https://img.shields.io/badge/Start_on-Killercoda-2496ED?style=for-the-badge)](https://killercoda.com/kubernauten/scenario/kubernetes-rbac-experienced)

**Summary:** Namespace Role, RoleBinding, ServiceAccount, and `kubectl auth can-i`.

### Import on Killercoda

1. Go to https://killercoda.com and sign in.
2. Create Scenario → Import from GitHub → `https://github.com/kubernauten/killercoda-scenarios`.
3. Select the `kubernetes-rbac-experienced` directory as the scenario folder.
