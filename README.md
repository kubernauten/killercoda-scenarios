# killercoda-scenarios

Available scenarios:

- [linux-fundamentals](linux-fundamentals/) – Linux basics: shell navigation, files, users, permissions, and packages.
- [k3s-01-setup](k3s-01-setup/) – Set up a lightweight k3s Kubernetes cluster and verify it is running.
- [kubernetes-1node](kubernetes-1node/) – Interact with a single-node Kubernetes cluster and list pods.
- [kubernetes-2node-multi-step-verification](kubernetes-2node-multi-step-verification/) – Work with a two-node cluster and pass multiple verification steps.
- [kubernetes-dashboard](kubernetes-dashboard/) – Deploy and access the Kubernetes Dashboard UI.
- [kubernetes-volumes](kubernetes-volumes/) – Use PersistentVolumes and PersistentVolumeClaims for storage.

## About Kubernauten

Kubernauten is a community of Kubernetes and cloud-native enthusiasts.
We learn together, we grow together.

These Killercoda scenarios exist to provide practical, hands-on labs for
learning Kubernetes and Linux concepts in a safe, browser-based environment.
They are used in workshops, meetups, and for self-paced learning.
