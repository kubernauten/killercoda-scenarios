## linux-process-monitoring-intermediate

**Summary:** `ps`, saving process output, background `sleep`, and `kill -0` on Ubuntu.

### Import on Killercoda

1. Go to https://killercoda.com and sign in.
2. Create Scenario → Import from GitHub → `https://github.com/kubernauten/killercoda-scenarios`.
3. Select the `linux-process-monitoring-intermediate` directory as the scenario folder.
