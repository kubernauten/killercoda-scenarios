## kubernetes-beginner-deployments

[![Start on Killercoda](https://img.shields.io/badge/Start_on-Killercoda-2496ED?style=for-the-badge)](https://killercoda.com/kubernauten/scenario/kubernetes-beginner-deployments)

**Summary:** Create a Deployment, inspect ReplicaSets and Pods, then scale replicas on a single-node kubeadm cluster.

### Import on Killercoda

1. Go to https://killercoda.com and sign in.
2. Create Scenario → Import from GitHub → `https://github.com/kubernauten/killercoda-scenarios`.
3. Select the `kubernetes-beginner-deployments` directory as the scenario folder.
